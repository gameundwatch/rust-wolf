# Environment Variables

## Backend (`backend/.env`)

| Variable         | Description              | Default |
|------------------|--------------------------|---------|
| `ALLOWED_ORIGIN` | CORS で許可するオリジン    | なし（必須） |

## Frontend (`frontend/.env`)

| Variable       | Description           | Default |
|----------------|-----------------------|---------|
| `VITE_API_URL` | バックエンドの Base URL | なし（必須） |
